# 武将卡牌裁剪与配置指南

## 📋 概述

本指南用于将四张预组整体图片中的武将卡牌裁剪出来，并配置到游戏应用中使用。

### 魏国（wei）

### 蜀国（shu）

- **武将**（14张）：孙乾、马云禄、雷铜、吴班、陈到、关羽、周仓、赵广、高翔、杨仪、关平、王平、吴毅

## 🚀 使用步骤

### 步骤 1：准备整体图片

将四张预组整体图片重命名为以下格式，并放到项目根目录的 `images/` 文件夹：

```
images/
├── wei_overall.png    # 魏国整体图片
├── shu_overall.png    # 蜀国整体图片
├── wu_overall.png     # 吴国整体图片
└── qun_overall.png    # 群雄整体图片
```

### 步骤 2：安装依赖

```bash
pip install Pillow
```

### 步骤 3：调整裁剪配置

打开 `crop_cards.py`，根据实际情况修改以下配置：

#### 3.1 卡牌尺寸配置

```python
CARD_SIZE = {
    "width": 180,   # 卡牌宽度（像素）
    "height": 250,  # 卡牌高度（像素）
    "spacing_x": 20, # 卡牌间水平间距
    "spacing_y": 20, # 卡牌间垂直间距
}
```

#### 3.2 布局配置

```python
LAYOUT_CONFIG = {
    "wei": {
        "start_position": (100, 400),  # 第一张卡牌的起始坐标
        "rows": [
            ["曹操"],  # 主帅
            ["乐进", "曹昂", "徐晃"],  # 第一行
            ["荀彧", "邓艾", "王双"],  # 第二行
            ...
        ]
    },
    ...
}
```

**如何确定起始坐标？**
1. 使用图片查看工具（如 Photoshop、GIMP）打开整体图片
2. 将鼠标移到第一张卡牌的左上角
3. 记录坐标值（X, Y）
4. 填入 `start_position`

### 步骤 4：运行裁剪脚本

```bash
python crop_cards.py
```

脚本会自动：
1. 读取四张整体图片
2. 按照配置裁剪出每张武将卡牌
3. 按武将名保存为 PNG 格式

输出目录结构：

```
images/cards/
├── wei/
│   ├── 曹操.png
│   ├── 乐进.png
│   ├── 曹昂.png
│   └── ...
├── shu/
│   ├── 刘备.png
│   ├── 孙乾.png
│   └── ...
├── wu/
│   ├── 孙尚香.png
│   ├── 朱治.png
│   └── ...
└── qun/
    ├── 貂蝉.png
    ├── 李儒.png
    └── ...
```

### 步骤 5：复制到项目资源目录

将裁剪后的图片复制到项目的静态资源目录：

```bash
# Windows PowerShell
Copy-Item -Path "images/cards/*" -Destination "src/main/resources/static/images/cards/" -Recurse -Force
```

或者手动复制所有文件到 `src/main/resources/static/images/cards/` 目录。

### 步骤 6：验证效果

1. 重启应用（如果正在运行）
2. 打开浏览器访问 `http://localhost:8080`
3. 开始模拟，检查武将卡牌图片是否正确显示

## 🔧 高级配置

### 自定义图片路径

如果某些武将的图片路径需要特殊指定，可以修改 `ImageConfig.java`：

```java
static {
    CARD_IMAGE_MAP.put("曹操", "/images/cards/wei/曹操.png");
    CARD_IMAGE_MAP.put("刘备", "/images/cards/shu/刘备.png");
    // ... 其他武将
}
```

### 使用不同格式

如果卡牌图片是 JPG 格式，修改 `ImageConfig.java`：

```java
private static final String CARD_IMAGE_SUFFIX = ".jpg";
```

## ⚠️ 常见问题

### Q1: 裁剪位置不准怎么办？

**A:** 调整 `LAYOUT_CONFIG` 中的 `start_position`，或者使用图片编辑工具（如 Photoshop）手动标记每张卡牌的坐标。

### Q2: 卡牌图片太小/太大？

**A:** 
- 方法 1：调整 `CARD_SIZE` 中的 `width` 和 `height`
- 方法 2：在裁剪后使用图片处理工具批量调整尺寸

### Q3: 如何使用 FFmpeg 批量裁剪？

**A:** 使用以下脚本：

```bash
# 安装 FFmpeg 后，创建裁剪脚本
# 示例：裁剪魏国卡牌

# 曹操
ffmpeg -i images/wei_overall.png -vf "crop=180:250:100:400" images/cards/wei/曹操.png

# 乐进
ffmpeg -i images/wei_overall.png -vf "crop=180:250:300:400" images/cards/wei/乐进.png

# ... 依此类推
```

### Q4: 如何批量添加卡牌名称标签？

**A:** 使用 FFmpeg 的 drawtext 滤镜：

```bash
ffmpeg -i card.png -vf "drawtext=text='曹操':fontsize=20:fontcolor=white:x=(w-text_w)/2:y=h-50" card_with_name.png
```

## 🎨 推荐图片规格

- **格式**：PNG（支持透明背景）或 JPG
- **尺寸**：180×250 或 360×500 像素（2倍图）
- **质量**：72-150 DPI
- **文件大小**：< 100KB/张

## 🔄 更新流程

如果需要重新裁剪：

1. 删除旧的裁剪图片：`rm -rf images/cards/*`
2. 调整 `crop_cards.py` 中的配置
3. 重新运行脚本：`python crop_cards.py`
4. 复制新图片到项目：`Copy-Item -Path "images/cards/*" -Destination "src/main/resources/static/images/cards/" -Recurse -Force`
5. 重启应用验证

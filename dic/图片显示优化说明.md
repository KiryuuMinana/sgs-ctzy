# 武将卡图片显示优化说明

## 📋 修改内容

### 1. 武将卡小图标位置调整 ✅

**问题**：武将卡的小图标（270×377px）在卡片中显示不完整，且位置偏上

**解决方案**：
- 将 `.card-image-area` 的 `overflow` 从 `hidden` 改为 `visible`，允许图片溢出显示
- 添加 `margin-top: 15px` 让图片向下移动约四分之一
- 将 `.general-card-inner` 的 `overflow` 从 `hidden` 改为 `visible`，防止裁剪

**修改的 CSS**：
```css
.card-image-area {
    overflow: visible;  /* 原：overflow: hidden */
    margin-top: 15px;   /* 新增：向下移动图片 */
}

.card-image-area img {
    height: auto;              /* 原：height: 100% */
    max-height: 100%;          /* 新增：限制最大高度 */
    object-fit: contain;       /* 原：object-fit: cover */
    object-position: bottom center;  /* 新增：图片底部对齐 */
}

.general-card-inner {
    overflow: visible;  /* 原：overflow: hidden */
}
```

### 2. 弹窗中图片完整显示 ✅

**问题**：点击武将卡后，弹窗中的图片（270×377px）显示不完整

**解决方案**：
- 将 `.skill-modal-image` 的固定高度改为自适应
- 设置 `min-height: 280px` 确保最小显示区域
- 将图片的 `object-fit` 从 `cover` 改为 `contain`，完整显示图片
- 设置 `overflow: visible` 允许图片完整显示

**修改的 CSS**：
```css
.skill-modal-image {
    height: auto;          /* 原：height: 180px */
    min-height: 280px;     /* 新增：最小高度 */
    overflow: visible;     /* 原：overflow: hidden */
}

.skill-modal-image img {
    height: auto;          /* 原：height: 100% */
    max-height: 400px;     /* 新增：限制最大高度 */
    object-fit: contain;   /* 原：object-fit: cover */
}
```

## 💡 技术说明

### object-fit 属性
- `cover`（原使用）：保持比例填充，可能裁剪
- `contain`（现使用）：保持比例完整显示，可能有空白

### overflow 属性
- `hidden`（原使用）：裁剪溢出内容
- `visible`（现使用）：显示溢出内容

### object-position 属性
- `bottom center`：图片底部居中对齐，确保重要内容（通常是人物脸部）可见
